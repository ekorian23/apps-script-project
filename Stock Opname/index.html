Form Stok Opname Karyawan

Form stock opname dilakukan setiap tahun untuk memastikan aset Ruangguru terdata dengan baik, dan wajib diisi.

Summary

1. Jika laptop anda Windows, Silahkan download file IP_SN.bat untuk memeriksa SN dan Mac Address. Jika file tidak bisa digunakan berikut panduan langkah manual yang dapat dilihat melalui link ini.

2. Jika laptop anda Macbook, Silahkan download file SN-MAC.app untuk memeriksa SN dan Mac Address. Jika file tidak bisa digunakan berikut panduan langkah manual yang dapat dilihat melalui link ini.
